# KERI 공공부문 AI 활용교육 패키지

본 저장소는 **한국교육연구소(KERI) AI활용교육 강사단**의 공공부문(교직원, 공무원, 준공무원 및 공공기관 임직원) 맞춤형 연수 설계, 교안 개발, 홍보 및 대외 행정 문서를 통합 관리하는 실무 패키지입니다.

공공 현장의 행정 비효율을 개선하고 디지털 역량을 제고하며, 대외 제안부터 강사 관리까지의 전 과정을 원스톱으로 처리할 수 있도록 공문서 행정 규격과 어조에 맞춰 구성되었습니다.
